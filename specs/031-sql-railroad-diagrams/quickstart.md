# Quickstart

## Local Development (Documentation)

Since this feature revolves around documentation, the quickstart involves previewing the documentation site locally.

1.  **Locate the Docs Setup:** Determine how the documentation in `docs/` is built (e.g., MkDocs, mdBook, Jekyll).
2.  **Serve Locally:** Run the appropriate local server command (e.g., `mdbook serve docs/` or `mkdocs serve`).
3.  **View Changes:** Navigate to `http://localhost:<port>/_docs/references/sql-commands/` to view the new category structure and verify the railroad diagrams render via the client-side JavaScript.
